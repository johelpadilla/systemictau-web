## Quick Start (Mac / Linux)

1. **Install Dependencies:**
   Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```
2. **Launch the App:**
   Simply double-click the `run.command` file, OR run:
   ```bash
   streamlit run app.py
   ```
